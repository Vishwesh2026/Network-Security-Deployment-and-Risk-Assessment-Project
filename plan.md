## Communication Plan

We meet weekly during tutorials to review progress, divide tasks, and resolve blockers. For day-to-day communication we use WhatsApp for quick messages and Zoom when we need to share screens or work through technical issues together.

Each meaningful task is followed by a GitHub commit so individual contributions are visible in the repository history. This ensures work is shared and reviewed, not concentrated in a single person's account.

Summary:

| Method               | Purpose                                         | Frequency                   |
| -------------------- | ----------------------------------------------- | --------------------------- |
| In-person (tutorial) | Progress review, task planning, problem solving | Weekly                      |
| WhatsApp             | Quick updates, file links, coordination         | As needed                   |
| Zoom                 | Screen sharing and joint configuration          | When required               |
| GitHub commits       | Track contributions and submit completed work   | After each significant task |
